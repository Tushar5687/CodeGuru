Basics of jQuery Events

jQuery simplifies event handling in JavaScript by providing methods to capture and respond to user interactions such as clicks, hovers, and key presses efficiently.

What are Events in jQuery?

Events are actions that occur in the browser, such as a button click, mouse hover, or text input. jQuery provides an easy way to bind these events to functions.

Common jQuery Event Methods

  • .click(): Binds a click event to an element.
  • .hover(): Attaches functions for mouse enter and leave events.
  • .keydown(): Triggers when a key is pressed.
  • .submit(): Executes when a form is submitted.

Example: Button Click Event

$(document).ready(function() {
  $("#btn").click(function() {
    alert("Button clicked!");
  });
});

Example: Hover Event

$(document).ready(function() {
  $(".hover-box").hover(
    function() { $(this).css("background-color", "yellow"); },
    function() { $(this).css("background-color", ""); }
  );
});
