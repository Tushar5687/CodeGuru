Basics of Selectors in jQuery

jQuery selectors are used to find and manipulate HTML elements based on their name, ID, classes, types, attributes, and more.

What are jQuery Selectors?

Selectors in jQuery are used to "select" and work with specific HTML elements. They are similar to CSS selectors but more powerful with additional filters and methods.

Common Selectors

  • $("element"): Selects all elements with the specified tag name.
  • $("#id"): Selects the element with the specified ID.
  • $(".class"): Selects all elements with the specified class.
  • $("[attribute=value]"): Selects elements with the specified attribute value.

Example: Select by ID

$(document).ready(function() {
  $("#example").css("color", "blue");
});

Example: Select by Class

$(document).ready(function() {
  $(".highlight").css("background-color", "yellow");
});

Example: Select by Attribute

$(document).ready(function() {
  $("[type='text']").val("Default Text");
});

Example: Select by Element

$(document).ready(function() {
  $("p").css("font-size", "18px");
});
