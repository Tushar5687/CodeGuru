JavaScript Functions

In this section, you will learn about JavaScript functions, how to create, use, and manipulate them. Functions are one of the most fundamental building blocks of JavaScript programming, allowing you to execute a block of code when called upon.

Introduction to Functions

In JavaScript, a function is a block of code designed to perform a particular task. Functions allow us to avoid repetition, reuse code, and make our programs more modular and organized. Functions can take inputs, perform operations, and return outputs.

Creating Functions

Functions can be created in JavaScript in several ways:

  • Function Declaration: This is the most common method of creating a function.
    • function greet() {
  console.log('Hello, World!');
}
  • Function Expression: A function can also be created by assigning it to a variable.
    • const greet = function() {
  console.log('Hello, World!');
};
  • Arrow Function: A more concise way to define functions introduced in ES6.
    • const greet = () => {
  console.log('Hello, World!');
};

Calling Functions

Once a function is defined, it can be called or invoked by using its name followed by parentheses.

  • Basic Function Call:
    • greet();
  • With Arguments: Functions can accept parameters or arguments, which are values passed into the function.
    • function greet(name) {
  console.log('Hello, ' + name);
}
greet('Alice'); // Output: Hello, Alice

Function Parameters and Return Values

Functions in JavaScript can accept parameters (also known as arguments), which allow you to pass data into the function. Functions can also return a value using the return keyword.

  • Function Parameters: Parameters are placeholders for the values that are passed into a function.
    • function add(a, b) {
  return a + b;
}
console.log(add(3, 5)); // Output: 8
  • Return Values: Functions can return values back to the caller.
    • function square(number) {
  return number * number;
}
console.log(square(4)); // Output: 16

Example: Working with Functions


function greet(name) {
  console.log('Hello, ' + name);
}

function add(a, b) {
  return a + b;
}

greet('Alice'); // Output: Hello, Alice
console.log(add(3, 5)); // Output: 8

Function Scope

Variables defined inside a function are local to that function, and cannot be accessed from outside the function. This is called local scope. However, variables defined outside a function are in the global scope, and can be accessed by any function.

  • Local Scope: Variables declared within a function are local to that function.
    • function example() {
  let x = 10; // x is local to this function
  console.log(x);
}
example();
  • Global Scope: Variables declared outside any function are accessible throughout the script.
    • let y = 20; // y is global
function example() {
  console.log(y); // y can be accessed inside the function
}
example();

Example: Scope in Action


let y = 20; // Global scope

function example() {
  let x = 10; // Local scope
  console.log(x); // Output: 10
  console.log(y); // Output: 20
}

example();
console.log(x); // Error: x is not defined (because it's local to the function)

Recursion

Recursion occurs when a function calls itself. Recursive functions are commonly used to solve problems that can be broken down into smaller subproblems, such as calculating factorials or traversing tree structures.

  if (n === 0) {
    return 1;
  }
  return n * factorial(n - 1);
}
console.log(factorial(5)); // Output: 120

Higher-Order Functions

In JavaScript, functions can accept other functions as arguments, return functions, or do both. These are called higher-order functions. For example, map(), filter(), and reduce() are higher-order functions in JavaScript.

  • Example of Higher-Order Function:
    • function greet(name, callback) {
  console.log('Hello, ' + name);
  callback();
}

greet('Alice', function() {
  console.log('How are you?');
});
