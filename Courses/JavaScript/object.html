JavaScript Objects

In this section, you will learn about JavaScript objects, how to create and manipulate them. Objects are key to JavaScript as they allow the organization of data in a structured way and are crucial in developing interactive web applications.

Introduction to Objects

In JavaScript, an object is a standalone entity, with properties and types. It is similar to real-life objects, like a car, that have properties (e.g., color, model) and methods (e.g., drive, stop). Objects in JavaScript are used to store collections of data and more complex entities.

Creating Objects

There are several ways to create objects in JavaScript:

  • Using Object Literals: This is the most common way to create objects. An object literal is enclosed in curly braces `{}`, and properties are defined using key-value pairs.
    • let car = { brand: 'Toyota', model: 'Camry', year: 2020 };
  • Using the Object Constructor: This uses the built-in Object constructor.
    • let car = new Object(); car.brand = 'Toyota'; car.model = 'Camry'; car.year = 2020;
  • Using Object.create(): This allows you to create a new object with a specific prototype object.
    • let car = Object.create(Object.prototype); car.brand = 'Toyota'; car.model = 'Camry'; car.year = 2020;

Object Properties and Methods

Properties of an object are values associated with the object. Methods are functions stored as properties within an object. Here is how you can access and define properties and methods:

  • Accessing Properties: You can access properties of an object using dot notation or bracket notation.
    • let car = { brand: 'Toyota', model: 'Camry', year: 2020 }; console.log(car.brand); // 'Toyota'
  • Adding Properties: You can add properties to an object dynamically.
    • car.color = 'Red';
  • Object Methods: Methods can be added to an object. These are functions defined inside the object.
    • let car = { brand: 'Toyota', model: 'Camry', year: 2020, displayInfo: function() { console.log(this.brand + ' ' + this.model); }};

Example: Working with Objects


let car = {
  brand: 'Toyota',
  model: 'Camry',
  year: 2020,
  displayInfo: function() {
    console.log(this.brand + ' ' + this.model);
  }
};

car.displayInfo(); // Output: Toyota Camry
car.color = 'Red';
console.log(car.color); // Output: Red

Modifying Objects

Objects in JavaScript can be modified by:

  • Changing Property Values: You can change the value of an existing property.
    • car.year = 2022;
  • Deleting Properties: You can delete properties using the delete keyword.
    • delete car.year;

Example: Modifying Objects


let car = {
  brand: 'Toyota',
  model: 'Camry',
  year: 2020
};

console.log(car.year); // 2020
car.year = 2022; // Modify the year property
console.log(car.year); // 2022

delete car.model; // Delete the model property
console.log(car.model); // undefined

Object Inheritance

JavaScript allows objects to inherit properties and methods from other objects. This is known as prototype inheritance. All objects in JavaScript have a prototype object, which can be used to share properties or methods among all instances of an object.

  • Prototype Chain: When trying to access a property or method, JavaScript first checks the object itself and then looks at its prototype, continuing up the prototype chain if necessary.
    • function Car(brand, model, year) {
  this.brand = brand;
  this.model = model;
  this.year = year;
}

Car.prototype.displayInfo = function() {
  console.log(this.brand + ' ' + this.model);
};

let myCar = new Car('Toyota', 'Camry', 2020);
myCar.displayInfo(); // Toyota Camry
