JavaScript Arrays

In this section, you will learn about JavaScript arrays, how to create them, and perform various operations on them. Arrays are one of the most important data structures in programming and are widely used to store and manage collections of data.

Introduction to Arrays

Arrays are used to store multiple values in a single variable. An array can hold elements of any data type, including numbers, strings, objects, and even other arrays. Arrays in JavaScript are dynamic, meaning their size can change as needed.

Creating Arrays

There are two main ways to create arrays in JavaScript:

  • Using the array literal syntax: `let fruits = ['Apple', 'Banana', 'Cherry'];`
  • Using the Array constructor: `let fruits = new Array('Apple', 'Banana', 'Cherry');`

Array Operations

JavaScript arrays come with built-in methods that allow you to perform various operations. Here are some commonly used methods:

  • push(): Adds an element to the end of an array.
  • pop(): Removes the last element from an array.
  • shift(): Removes the first element from an array.
  • unshift(): Adds an element to the beginning of an array.
  • length: Returns the number of elements in an array.

Example: Working with Arrays

        
          let fruits = ['Apple', 'Banana', 'Cherry'];
          console.log(fruits[0]); // Apple
          
          fruits.push('Orange'); // Add Orange to the end
          console.log(fruits); // ['Apple', 'Banana', 'Cherry', 'Orange']
          
          fruits.pop(); // Remove the last element
          console.log(fruits); // ['Apple', 'Banana', 'Cherry']

Example: Array Methods

Let's see an example that demonstrates various array methods:

        
          let numbers = [10, 20, 30];
          console.log(numbers.length); // 3
          
          numbers.shift(); // Removes the first element (10)
          console.log(numbers); // [20, 30]
          
          numbers.unshift(5); // Adds 5 to the beginning
          console.log(numbers); // [5, 20, 30]
